# Maths for AI

Bienvenue sur le dépôt du projet de recherche **Maths for AI**, un recueil des fondements mathématiques indispensables pour comprendre et développer l'Intelligence Artificielle.

## Contenu

Ce projet regroupe 5 chapitres majeurs abordant les concepts clés :
1. Algèbre Linéaire & Tenseurs
2. Optimisation
3. Probabilités & Information
4. Calcul Différentiel
5. Géométrie des Données

## Déploiement

La documentation est générée avec MkDocs (Material) et les chapitres mathématiques sont écrits en LaTeX. 
Une action GitHub s'occupe de compiler les fichiers LaTeX en PDF et de déployer le site web sur GitHub Pages à chaque `push`.

## Comment contribuer ?

Vous souhaitez ajouter des démonstrations mathématiques, corriger une erreur ou proposer une nouvelle approche ? Les contributions sont les bienvenues !

1. **Forkez** le dépôt.
2. **Créez une branche** pour votre fonctionnalité (`git checkout -b feature/ma-demo`).
3. **Modifiez** les fichiers LaTeX dans le répertoire `themes/` (assurez-vous d'utiliser les environnements LaTeX existants comme `Theorem`, `Definition`, `Proof`).
4. **Commitez** vos changements (`git commit -m 'Ajout de la démonstration du théorème X'`).
5. **Poussez** vers la branche (`git push origin feature/ma-demo`).
6. **Ouvrez une Pull Request**.

Merci pour votre aide dans la construction de cet écosystème !
